## What We Do NOT Do

The platform does **not** intentionally engage in:

| Prohibited Activity | Description |
| --- | --- |
| **Phishing** | No deceptive attempts to obtain user credentials |
| **Malicious Wallet Schemes** | No hidden wallet-draining mechanisms |
| **Unauthorized Movement** | No movement of assets without user consent |
| **Deceptive Approvals** | No hidden or misleading approval requests |
| **Exploitative Contracts** | No malicious smart contract behavior |

## Intended User Base

The platform is intended for users seeking professionally managed Web3 treasury allocation infrastructure:

- **Digital asset holders** - Users with USDC, USDT, or other supported assets
- **Web3-native users** - Users familiar with wallet connections and blockchain interactions
- **Treasury participants** - Organizations or individuals managing digital asset treasuries
- **Yield seekers** - Users seeking managed digital asset yield opportunities

<Warning>
  The platform is **not** intended to function as:

  - Malicious software
  - Exploit infrastructure
  - Phishing infrastructure
  - Deceptive wallet tooling
</Warning>
